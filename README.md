# Andromeda Brief Builder

An interactive copy-brief generator for high-differentiation direct-response copy.
Pick one variant from each building block (Audience, Mechanism, Hook, Failed Solutions, etc.),
hit **Generate**, and copy the assembled prompt to paste into an AI copywriter.

The blocks are designed so that changing the Tier-1 blocks (Audience, Mechanism, Hook,
Failed-Solutions vocabulary) produces a genuinely different Meta/Andromeda audience vector —
not the same copy in a new story jacket.

## Use it
**Live:** see the GitHub Pages URL in the repo's About / Settings → Pages.
Or open `index.html` locally in a browser.

## Edit it
It's a single self-contained file (`index.html`) — inline HTML/CSS/JS, no dependencies.
Edit and push; GitHub Pages redeploys automatically.

Part of the Keaps direct-response copy system.
